A comprehensive fitness web application designed to help gym-goers understand equipment, practice proper form, and generate intelligent workout plans based on their goals.

## Features

*   **Intelligent Workout Generator:** Generates personalized routines based on your Age, Weight, Experience, and Goal (Muscle Gain, Weight Loss, Strength).
*   **Machine Encyclopedia:** Detailed guide for gym machines with "How-to" instructions and targeted muscle groups.
*   **Safety First:** Special logic for users 50+ to ensure safe workout recommendations.
*   **Progressive Web App (PWA) Capable:** Built with modern web tech for a smooth mobile experience.
*   **Admin Dashboard:** easy way to add new machines.

## Tech Stack

*   **Frontend:** React.js (Vite)
*   **Styling:** Tailwind CSS
*   **State Management:** React Context API

## Getting Started

1.  Clone the repository:
    ```bash
    git clone https://github.com/kiranYadav-Bob/GymGuide-Pro.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```
